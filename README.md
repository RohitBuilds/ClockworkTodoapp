Clockwork Todo App 

A modern full-stack task management application designed to help users organize, track, and manage daily tasks efficiently. Clockwork Todo provides a clean and responsive interface where users can create tasks, set reminders, track completion status, and manage productivity in a structured way.
This project demonstrates full-stack development skills, combining a dynamic frontend with a scalable backend API and a database for persistent task storage.

Features

User Authentication
Secure user signup and login
Session-based authentication
Protected routes for authenticated users

Task Management

Create new tasks with title and description
Edit existing tasks
Mark tasks as completed or incomplete
Delete individual tasks
Clear all tasks

Task Reminder System

Add due date and time for tasks
Visual indicators for upcoming tasks
Helps users stay organized and productive
Task Status Tracking

Toggle tasks between Done / Undo

Completed tasks display a visual indicator
Easy-to-read task organization

Modern UI

Clean and responsive interface
Smooth user interactions
Designed for better productivity
